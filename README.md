# Offline Document Search Engine

This Python script demonstrates a simple offline document search engine. It indexes text content from provided documents and allows users to search for keywords without an internet connection. The engine builds an inverted index for efficient retrieval.

## Language

`python`

## How to Run

Save the code as `offline_search.py` and run it from your terminal using `python offline_search.py`.
